# face-detection

Write a python script that can detect faces in an image and save all the faces in a folder

Hint: <a href="https://raw.githubusercontent.com/opencv/opencv/master/data/haarcascades/haarcascade_frontalface_default.xml">haar cascade classifier</a> can be used to detect the faces. It returns coordinates of the faces, using those faces can be saved as a file.

Installations: OpenCV

Downloads: <a href="https://raw.githubusercontent.com/opencv/opencv/master/data/haarcascades/haarcascade_frontalface_default.xml">haarcascade_frontalface_default.xml</a>
